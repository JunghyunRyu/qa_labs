```

│ (자동)       │    │ (수동)       │    │ (수동)       │    │ (API/SQL)    │
└──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘
```

**문제 생성 명령어:**
```bash
# Backend 가상환경 활성화 후
cd backend
.\venv\Scripts\Activate.ps1

# AI 문제 생성 API 호출 (PowerShell)
$body = @{
    goal = "경계값 분석을 평가하는 QA 코딩 테스트 문제 생성"
    language = "python"
    testing_framework = "pytest"
    skills_to_assess = @("boundary value analysis", "edge cases")
    difficulty = "Easy"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://localhost:8000/api/admin/problems/ai-generate" -Method Post -Body $body -ContentType "application/json"
```

## 🚀 Quick Start 가이드

### 오늘 바로 시작하기

```powershell
# 1. 프로젝트 폴더로 이동
cd C:\jhryu\qa_labs

# 2. Backend 가상환경 활성화
cd backend
.\venv\Scripts\Activate.ps1

# 3. Docker 서비스 시작 (PostgreSQL, Redis)
cd ..
docker-compose up -d db redis

# 4. DB 마이그레이션 실행
cd backend
alembic upgrade head

# 5. Backend 서버 시작
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 6. (새 터미널) Frontend 시작
cd C:\jhryu\qa_labs\frontend
npm run dev

# 7. 브라우저에서 확인
# http://localhost:3000 (Frontend)
# http://localhost:8000/docs (Backend API 문서)
```
